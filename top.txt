2.) Don't use global variables (usually best to use let over var)
3.) Use comments to help explain your code (try to make code simple rather than overly complex however)
4.) Do not overuse DOM manipulation
5.) Stick to one task per one function

9.) use template strings for variable concatenations
10.) include error handling in your code
